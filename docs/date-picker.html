属性

属性 说明 类型 默认值
start-date 限制可选开始时间 String, Date
end-date 限制可选结束时间 String, Date
disabled-date 自定义方法限制可选时间。传入日期，返回true表示禁用该日期 Function(date)
v-model 绑定的数据。范围选择时传入Array，Array[0]为开始日期,Array[1]为结束日期 String, Date, Array
type 选择器类型。可选值：year, month, date, datetime, daterange, datetimerange, time String 'date'
format 日期格式。如果不设置，默认会根据type进行判断。 String 根据type进行默认配置
clearable 鼠标移入显示清空按钮 Boolean
placeholder placeholder String
