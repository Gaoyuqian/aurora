基础使用方法

分页组件默认每页10条数据。show-total控制是否显示总数。

page={{page}} itemCount={{itemCount}}

位置控制

page={{page}} itemCount={{itemCount}}

显示更多分页控制

page={{page}} itemCount={{itemCount}}

覆盖原有pagesize选项

page={{page}} itemCount={{itemCount}}

pageSize控制

page={{page}} itemCount={{itemCount}} pageSize={{pageSize}}

属性

属性 说明 类型 默认值
align 可选值：'left','center','right' String 'left'
v-model 绑定的当前页 Number 1
item-count 总条目数 Number 0
page-size-options 每页可显示的数量 Array [10, 20, 50]
page-size 绑定每页可显示的数量。发生变化时，将触发 page-size-change 事件 Number page-size-options属性的第一个元素值 或 10
show-page-size-control pagesize是否可改变 Boolean false
show-page-control 是否显示页数控制输入框 Boolean false
show-total 是否显示总页数 Boolean false

事件

名称 说明
page-size-change page-size 改变时触发
