属性

属性名 说明 类型 默认值
type 按钮类型。可选值包含primary、info、success、danger、default、warning、link String 'default'
size 按钮大小。可选值包含large、default、small、mini 四种大小 String 'default'
href 跳转链接，如果设置链接将使用a标签 String
target a标签的href属性a设置href后该值才生效。 String
loading 是否开启loading 状态 Boolean false
disabled 是否禁用按钮 Boolean false
icon 按钮图标（图标只能加在按钮左边，如果需要加载按钮右边，请使用au-icon自行添加） String
nativeType 原生按钮type属性 String 'button'
block 是否是块级按钮，将填满整个父元素宽度 Boolean false
disableOnClick 是否在点击之后禁用按钮 Boolean false
