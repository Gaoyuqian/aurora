属性

参数 说明 类型 默认值
type 按钮的样式 (可选值: primary, info, success, warning, danger, default, link) String 'default'
size 按钮的大小 (可选值: large, default, small, mini) String 'default'
href 按钮是否是一个链接 String null
target 如果提供了href, 可指定链接的target String null
loading 是否显示加载中的图标 Boolean false
disabled 是否禁用按钮 Boolean false
heading-icon 显示文字前方的icon String null
trailing-icon 显示文字后方的icon String null
native-type 原生type属性 String 'button'
autofocus 是否自动聚焦 Boolean false
block 是否显示块化按钮 Boolean false
