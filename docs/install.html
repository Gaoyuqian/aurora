安装

Aurora是基于Vue 2.2.0版的组件库, 请使用Vue 2.2.0以上的版本

Github地址

https://github.com/MeiliFE/Aurora

安装font-awesome

Aurora目前使用了fontawsome图标库, 由于未来会使用自己的图标库, 所以并没有集成fontawsome, 用户需要自行添加fontawesome字体库 http://fontawesome.io/

安装Aurora

请下载aurora的js, css文件, 并引入进项目中

  • 下载 aurora.js
  • 下载 aurora.css
<!-- 引入样式 -->
<link rel="stylesheet" href="/your/path/aurora.css">
<!-- 引入组件库 -->
<script src="/your/path/aurora.js"></script>

确保vue.js在Aurora.js前引入，Aurora自动执行Vue.use()
